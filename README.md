# 🥤 Organic Juice Café Website

## 📌 Project Overview

This is a **frontend website project** for an Organic Juice Café built using **HTML, CSS, and JavaScript**.
The website showcases fresh juice products, trending items, and allows users to interact with features like **Add to Cart** and **Load More**.

---

## 🚀 Features

* 🏠 Home section with attractive hero banner
* 🍹 Juice menu with horizontal scrolling
* 🔥 Trending juices section
* 🛒 Add to Cart functionality
* ➕ Load More button (shows additional products)
* 📱 Responsive design (mobile & desktop)
* 🧾 About section (café details & mission)
* 📞 Contact section with form

---

## 🛠️ Technologies Used

* HTML5
* CSS3 (Flexbox, Grid, Responsive Design)
* JavaScript (DOM manipulation)
